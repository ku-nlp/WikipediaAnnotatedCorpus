# S-ID:wiki00011396-00 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:30.00000
アルカンとは、一般式　で表される鎖式飽和炭化水素である。
# S-ID:wiki00011396-01 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:10.00000
メタン系炭化水素、パラフィン系炭化水素や脂肪族化合物とも呼ばれる。
# S-ID:wiki00011396-02 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:80.00000
炭素数が大きいものはパラフィンとも呼ばれる。
# S-ID:wiki00011396-03 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:38.00000
アルカンが置換基となった場合、一価の置換基をアルキル基、二価の置換基をアルキレン基と呼ぶ。
# S-ID:wiki00011396-04 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:60.00000
環状の飽和炭化水素はシクロアルカンと呼ばれる。
