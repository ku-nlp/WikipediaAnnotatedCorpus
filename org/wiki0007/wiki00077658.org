# S-ID:wiki00077658-06 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:58.00000 MOD:2023/08/29 MEMO:
アイオワ州法によって、アメリカ合衆国大統領選挙の前哨戦である大統領候補指名党員選挙を、全国に先駆けて行うことが定められている。
# S-ID:wiki00077658-07 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:78.00000 MOD:2023/08/29 MEMO:
したがって、アイオワ州党員選挙は大統領選挙の初戦としての位置付けにあり、大統領選挙の際には世界的に注目される。
# S-ID:wiki00077658-08 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:138.00000 MOD:2023/08/29 MEMO:
なお、アイオワ州党員選挙にて敗北した候補者が大統領に就任した例は少なく、「アイオワを制する者が大統領選挙を制する」とも言われている。
