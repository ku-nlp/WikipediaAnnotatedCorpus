生物学的には、果実とは被子植物の、その中に種子を含む構造のことである。
# S-ID:wiki00003719-01 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:136.00000
被子植物の種子は子房の中で成熟するから、子房が果実になる部分であり、おおかたすべての被子植物の種子は果実に入っている。
# S-ID:wiki00003719-02 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:124.00000
果実と聞けば、肉質で食せるものとの印象を与えるのは、そのような果実を形成する種が少なからずあってそれがヒトにとって重要な食料であったことによる。
# S-ID:wiki00003719-03 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:174.00000
このような果実は、植物の繁殖戦略として、動物の食料になる部分を種子の周りに発達させ、食べられることで動物の体内を通じて種子の散布をおこなうという目的のために進化したものであると考えられる。
# S-ID:wiki00003719-04 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:109.00000
こうした被食用の進化は、被食される対象に応じてそれぞれ進化方向が異なり、それに対応した果実を実らせるようになっている。
# S-ID:wiki00003719-05 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:173.00000
たとえば鳥類に被食されるために進化したものにおいては樹木の上方に結果し、大きさは小型のものが多い鳥類に合わせ数mm程度であり、視覚に訴えかけるために目立つ色彩をしている。
# S-ID:wiki00003719-06 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:121.00000
逆に哺乳類は鳥類に対し嗅覚が強いため、香りを強く放つよう進化したものが多い。
# S-ID:wiki00003719-07 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:58.00000
このほか、目立つ色彩と強い香りを併せ持ち、鳥類・哺乳類双方に捕食されやすいように進化した果実も存在する。
# S-ID:wiki00003719-08 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:160.00000
人類はこうした果実を食料とするため、畑で果菜を、果樹園で果樹とその果実である果物を栽培するが、これも広義においてはこの流れの中に属するといえる。
# S-ID:wiki00003719-09 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:70.00000
逆にそのような部分を持たない場合、一般には果実と認識されない例も多い。
# S-ID:wiki00003719-10 KNP:5.0-6a1f607d DATE:2022/04/27 SCORE:69.00000
後述の痩果は、往々にして種子と同一視される。
